Este aplicativo de escritorio fue creado para la asignatura de Taller de Programacion Aplicada.
Esta desarrollado en python y utiliza la biblioteca wxPython para la interfaz grafica.
Implementa cuatro funcionalidades: Ingresar Cliente, Buscar Cliente, Eliminar Cliente e Imprimir Clientes.

El objetivo es mostrar el concepto de estilo arquitectural MVC de manera simple. Los beneficios son:

1. Separacion de intereses (Modularizacion)
2. Accesos entre capas bien definidas
3. Posibilidad de cambiar la interfaz de usuario (Vista) sin comprometer las demas capas. En 
este ejemplo, existe una interfaz grafica y otra en modo consola, manteniendo las funcionalidades.
